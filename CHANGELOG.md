# Changelog

All notable changes to `phumtech/laravel-pdf-compress` will be documented in this file.

## 1.0.2 - 2026-04-29

- Added support for bundled qpdf binaries in the project root for Windows users
- Improved binary detection logic to automatically find local binaries
- Updated configuration with local path documentation

## 1.0.0 - 2026-04-27

- Initial release
- Support for qpdf and Ghostscript engines
- Fluent API for PDF compression
- Laravel Storage integration
- Quality presets (low, balanced, high)
